SO101-Nexus uses MuJoCo as its simulation backend. A MuJoCo Warp backend for GPU-parallel simulation is planned but not yet implemented.

## Task Availability

All five tasks run on the SO-101 arm:

| Task | Robot |
|------|-------|
| PickLift | SO-101 |
| PickAndPlace | SO-101 |
| Reach | SO-101 |
| LookAt | SO-101 |
| Move | SO-101 |

## Vectorization

MuJoCo environments are vectorized through Gymnasium's standard `gym.vector.SyncVectorEnv` wrapper, which runs multiple environment instances on CPU. The planned MuJoCo Warp backend will add GPU-parallel simulation.

## Objects

MuJoCo supports `CubeObject`, `YCBObject`, and `MeshObject`. All environments use the same composable observation components (`observations` parameter). See [Observations](/docs/concepts/observations-and-camera-modes) for details.

For details on object types, see the [Objects API reference](/docs/api/objects). For observation configuration, see [Observations](/docs/concepts/observations-and-camera-modes).
