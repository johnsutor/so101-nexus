# Core Package

`so101_nexus_core` provides shared types, configuration classes, scene objects, asset helpers, and the environment registry used by all backends. It is installed automatically as a dependency of any backend package.

```python
import so101_nexus_core
```

## Exports by Category

### Configuration Classes

All configuration classes are dataclasses with sensible defaults. See [Configuration Classes](/docs/api/configs) for full reference.

| Class | Purpose |
|-------|---------|
| `CameraConfig` | Camera resolution and wrist camera randomization ranges |
| `RobotConfig` | Rest pose, grasp force threshold, velocity threshold |
| `RobotCameraPreset` | Named preset combining camera parameters |
| `RewardConfig` | Reward component weights with a `compute()` method |
| `EnvironmentConfig` | Base environment parameters (spawn region, camera mode, episode length) |
| `PickConfig` | Extends `EnvironmentConfig` for pick/lift tasks |
| `PickAndPlaceConfig` | Extends `EnvironmentConfig` for pick-and-place tasks |

### Scene Objects

See [Scene Objects](/docs/api/objects) for full reference.

| Class | Description |
|-------|-------------|
| `SceneObject` | Abstract base for all scene objects |
| `CubeObject` | A colored cube with configurable size and mass |
| `YCBObject` | A YCB benchmark object loaded from mesh assets |
| `MeshObject` | A custom mesh object (MuJoCo only) |
