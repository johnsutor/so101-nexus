---
title: Installation
description: Install SO101-Nexus with pip or from source.
---

Python **>=3.12** is required.

## Install with pip

```bash
pip install so101-nexus
```

## Try without installing

If you only want to try the teleop recorder, [uv](https://docs.astral.sh/uv/) can run it in an ephemeral environment with no permanent install:

```bash
uvx --from "so101-nexus[teleop]" so101-nexus teleop
```

See the [teleoperation docs](/docs/teleoperation/overview) for hardware requirements, troubleshooting, and full usage.

## Install from Source

Clone the repository, then sync with `uv`:

```bash
git clone https://github.com/johnsutor/so101-nexus.git
cd so101-nexus
uv sync
```

## Verify Installation

```python
import so101_nexus.mujoco
import gymnasium as gym

env = gym.make("MuJoCoPickLift-v1")
print("Installation OK")
env.close()
```
