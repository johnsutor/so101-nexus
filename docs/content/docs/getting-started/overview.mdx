SO101-Nexus is a single installable library, `so101-nexus`. Install it with `pip install so101-nexus`.

The shared API (configuration, rewards, observations, scene objects, teleop, and the LeRobot adapter) lives at the top level under `so101_nexus`. The MuJoCo backend lives at `so101_nexus.mujoco`.

Importing the MuJoCo backend registers its environments with Gymnasium:

```python
import so101_nexus.mujoco   # registers MuJoCo envs
```

After importing, create environments with `gymnasium.make()`.

## Backend

MuJoCo is the current simulation backend. It provides all five tasks (PickLift, PickAndPlace, Reach, LookAt, Move) on the SO-101 arm, fast single-environment simulation with a live viewer, and custom mesh objects (`MeshObject`). A MuJoCo Warp backend for GPU-parallel simulation is planned.
